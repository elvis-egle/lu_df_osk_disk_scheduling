About the project

This is a simple HTML5/CSS3/JS website group project, where the main goal is to visualize disk scheduling algorithms. The animated point-to-point graph idea comes from Disk Scheduling Algorithms tutorial page, which belongs to Illinois Institute of Technology.
This project was given as an assignment in "Operating System concepts" course taught at University of Latvia, Faculty of Computing.

Technologies used

  • jQuery 3.1.1
  • Bootstrap v3.3.7
  • bootstrap-select
  • HTML5/CSS3/JS
